# Bill Spltting App

This is an app on web for bill splitting purpose. Have you ever encounter the situation where a group of friends hanging out together, sharing foods, car rental, tickets, etc. Some of the guys might pay for others at the moment first and collect money afterwards. Sometimes it's quite complicated to calculate the correct amount, espeacially when not only one person paying first for others. I've been there, so it occurs to me that it would be great if I create an app to solve this problem not only for me but any one who struggles in the situation.
